## Compiler Usage
There are three commands (shell scripts) for invoking the eps compilers:

###  ec [-c -C -a -s -p -t -f] file.e 

  - c: Generate c-code 
  - C: Generate code for arry index checking
  - a: Print annotated parse tree on standard output
